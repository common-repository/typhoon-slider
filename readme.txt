An HTML slider to do it all. 

== Description ==
This is an easy to use slider which you can use to create any kind of slides. You can create simple image slider, video slider, youtube slider or whatever you wish to do with this.

There are many other options like: slide, fade, set timing, etc.

== Installation ==
1. Go to Dashboard > Plugins > Add Plugin. 
1. Search for Typhoon slider.
1. Install and Activate the plugin
1. By now you might have an a menu option named \"Slider\", click on \"Add New\" under that.
1. Add slides
1. Use shortcode to add slider on the front end.
